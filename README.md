# DiffractionMeasure
1024个点阵采集衍射和干涉光强信号后，通过计算初始干涉位置，以及计算第一级衍射环的内外径长度，采用光学理论，计算激光的精确波长和线宽。</br>
文档是主要的实现部分，信号处理大部分细节省略掉了，本文档包含三个部分：</br>
1.采样的基础数据Data文件夹；</br>
2.用于VisualC++的处理代码；</br>
3.用在Labview接口上的（CNode）C处理代码。</br>
